# User Profile Card

A simple, responsive User Profile Card built with semantic HTML and pure CSS, created for a Web Programming assessment (HTML Structure & CSS Styling).

## 🔗 Live Preview
Open `index.html` in any browser to view the card.

## 📁 Files
- `index.html` — Semantic markup for the profile card (avatar, name, bio, stats, follow button)
- `style.css` — Styling for layout, typography, avatar, stats, and button interactions

## ✨ Features
- Fully responsive card centered both horizontally and vertically
- Circular avatar image with clean border
- Flexbox-based social stats (Followers / Following)
- Modern call-to-action button with hover effect
- Clean sans-serif typography and visual hierarchy

## 🛠️ Built With
- HTML5 (semantic tags: `<article>`, `<ul>`, `<button>`)
- CSS3 (Flexbox, box-shadow, border-radius, transitions)
